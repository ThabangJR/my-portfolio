## Features

- **Animated page loader** — ink-stamp TM initials with spinning ring
- **Magnetic buttons** — CTA buttons physically pull toward the cursor
- **Scroll-triggered counters** — stats count up from zero on scroll
- **Dark / light mode** — full theme toggle, persists across sections
- **Floating "Available for hire" stamp** — hero section badge
- **Timeline section** — visual career journey from school to now
- **Resume download** — one-click PDF download for recruiters
- **Copy email** — clipboard copy with toast notification
- **Keyboard navigation** — press `1`–`6` to jump sections, `?` for hint
- **Scroll reveal animations** — every section fades in on scroll
- **Skill pills** — real devicon brand icons with hover brand-color fill
- **Project cards** — 3D tilt + click-through to live sites
- **Contact form** — with animated success state
- **Fully responsive** — mobile hamburger menu, fluid spacing

## Tech Stack

| Tool | Purpose |
|------|---------|
| [React 18](https://react.dev) | UI library |
| [TypeScript](https://www.typescriptlang.org) | Type safety |
| [Vite](https://vitejs.dev) | Dev server & bundler |
| [Devicons CDN](https://devicon.dev) | Skill icons |
| [Google Fonts](https://fonts.google.com) | Playfair Display + Instrument Sans |

No UI framework. No Tailwind. Pure CSS-in-JS and a single global stylesheet.


## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `1` | Jump to About |
| `2` | Jump to Skills |
| `3` | Jump to Work |
| `4` | Jump to Journey |
| `5` | Jump to Resume |
| `6` | Jump to Contact |
| `?` | Toggle keyboard hint overlay |
